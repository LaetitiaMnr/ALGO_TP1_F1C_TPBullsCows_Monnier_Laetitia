- Jeu Bulls & Cows -

Pour gagner, il vous faut deviner le mot caché.

Si une lettre existe dedans et qu'elle est bien placée, un point est ajouté dans les Bulls.
Si une lettre existe mais qu'elle est mal placée, un point est ajouté dans les Cows.
Si la lettre n'existe pas, un 0 est affiché dans les Bulls & Cows.

Aidez-vous des Bulls & Cows afin de deviner le mot caché !
